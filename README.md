# Amazon-Sales-EDA-project

Amazon Sales Exploratory Data Analysis (EDA)

An end-to-end Exploratory Data Analysis (EDA) project that analyzes Amazon product data using Python to uncover pricing trends, customer ratings, discounts, and category-wise insights.

---

# 📌 Project Overview

The objective of this project is to clean, preprocess, and analyze Amazon product data to extract meaningful business insights. The analysis includes data cleaning, feature engineering, visualization, and answering business-related questions using Python.

---

# 🎯 Objectives

- Clean and preprocess raw Amazon product data
- Perform exploratory data analysis (EDA)
- Analyze pricing and discount patterns
- Study customer ratings and review counts
- Identify top-performing product categories
- Generate business insights through visualizations

---

# 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
- Git & GitHub

---

# 📂 Dataset

The dataset contains Amazon product information, including:

- Product Name
- Category
- Actual Price
- Discounted Price
- Discount Percentage
- Rating
- Rating Count
- Product Link
- Product Description

---

# 📋 Project Workflow

### 1. Data Loading

- Imported dataset using Pandas
- Inspected dataset structure
- Verified data types

### 2. Data Cleaning

- Removed duplicate records
- Handled missing values
- Converted prices into numeric format
- Converted discount percentage to numeric
- Cleaned rating values
- Standardized column names

### 3. Feature Engineering

Created new columns such as:

- Price Saved
- Discount Amount
- Rating Category
- Price Range

### 4. Exploratory Data Analysis

Performed analysis on:

- Product distribution
- Categories
- Ratings
- Discounts
- Prices
- Reviews

### 5. Data Visualization

Created charts including:

- Bar Charts
- Histograms
- Pie Charts
- Box Plots
- Line Charts
